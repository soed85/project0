How to create a new repository locally and push it to Github!

Mac: Open the terminal and go to the folder where your files are stored. Execute 'git init'. Go to Github.com and create a new repository. Copy link and execute this link with following command in the terminal 'git add remote origin link'. Add and commit relevant files. Execute 'git push'.

Terminal Description
git init Desc. 1
git add remote origin Desc. 2
git add Desc. 3
git commit Desc. 4
git push Desc. 5
